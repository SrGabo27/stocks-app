# Stocks App
Real-time stocks at your fingertips.

## Installation
Clone repository
```console
git clone https://github.com/SrGabo27/stocks-app.git

```
Enter the directory
```console
cd stocks-app/
```

Install Dependencies
<details>
<summary>npm</summary>

```
npm install
```
</details>
<details>
<summary>yarn</summary>

```
yarn
```
</details>
<details>
<summary>pnpm</summary>

```
pnpm install
```
</details>
